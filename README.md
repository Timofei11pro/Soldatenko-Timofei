# Weather Diary

**Автор:** [Солдатенко Тимофей]

### Описание
Приложение для ведения личного дневника погоды. Позволяет фиксировать дату, температуру и условия, сохраняя историю в формате JSON.

### Как запустить
1. Установите Python 3.x.
2. Скачайте файл `weather_diary.py`.
3. Запустите через терминал: `python weather_diary.py`.

### Использование
- Введите данные в поля и нажмите "Добавить".
- Используйте кнопку фильтра для поиска теплых дней (>10°C).
- Данные автоматически сохраняются в `weather_data.json`.
